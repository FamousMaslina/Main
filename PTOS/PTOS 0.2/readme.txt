PTOS is based on OP2 Version 0.8.4. Its API, BIOS, hardware identifier and Version File are all ported.
PTOS can run all OP2 files, due to its ported API.
PTOS is much more lighter: No extra apps, only the basics, and also many similarties with OP2: Like help, dvcman and so on.
